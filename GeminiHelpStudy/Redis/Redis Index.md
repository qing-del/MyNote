## 基本概念
- [Redis 官网](https://redis.io)
- 与`MySQL`的**区别**是，`Redis`使用的是类似于哈希表的那种`key=value`键值对的**内存**存储方式
> [!warning] ⚠注意
> `Redis`官方没有`Windows`的版本，只有`MacOS`和`Linux`的
> 所以`Windows`版本的`Redis`都是别人来开发和维护的
- **`Windos`下载**：[Release Redis for Windows 5.0.14.1](https://github.com/tporadowski/redis/releases/tag/v5.0.14.1)
> [!warning] ⚠ 此已经是最新的 `Windows` 的 `Redis` 版本了
> 同时可以来这里下载`Redis`可视化工具[Redis Desktop Manager](https://github.com/qishibo/AnotherRedisDesktopManager)
