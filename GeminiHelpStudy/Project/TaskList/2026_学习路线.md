## 📅 Phase 2: 内功心法修炼 (3月中旬 - 4月中旬)
**目标**: 建立知识体系，能通过技术面试的连环追问

### 🐬 MySQL 深度 (不背不舒服)
- [ ] **索引原理**: B+树结构、聚簇 vs 非聚簇、回表、覆盖索引
- [ ] **索引失效**: 背诵 5 种常见失效场景，并自己写 SQL 验证 (Explain)
- [ ] **事务与锁**: ACID、MVCC 原理、行锁 vs 表锁、间隙锁 (Gap Lock)
- [ ] **实战题**: "一张表 1000 万数据，怎么做分页查询优化？"

### ⚡ Redis 缓存 (高并发三板斧)
- [ ] **缓存异常**: 穿透、击穿、雪崩 (结合苍穹外卖代码讲解决方案)
- [ ] **数据一致性**: 先删缓存还是先更库？延时双删？(理解最终一致性)
- [ ] **分布式锁**: `setnx` 原理，Redisson 看门狗机制 (WatchDog)

### 🧵 JUC 并发编程 (难点)
- [ ] **线程池**: 7 大参数详解，拒绝策略，阿里规范为什么不让用 Executors
- [ ] **ThreadLocal**: 原理与内存泄漏 (结合你的 LifeGame 鉴权模块复习)
- [ ] **锁机制**: `synchronized` 升级过程，`ReentrantLock` 原理 (AQS)
