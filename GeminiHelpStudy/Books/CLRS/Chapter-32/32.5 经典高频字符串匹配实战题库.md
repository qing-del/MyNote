## 1. 原汁原味的基石大比拼：实现神级引擎 (Easy/Medium)

在一切极其花俏的图卷前。所有大厂（例如阿里字节极爱在此关卡考察底层轮子构造原性）要求你手撕的必定是那个引擎核心。不允许调用内部函数（也就是禁止你用如 C 里的 `strstr` 或者 Java 里的 `indexOf`）。

### [LeetCode 28. 找出字符串中第一个匹配项的下标 (Find the Index of the First Occurrence in a String)](https://leetcode.cn/problems/find-the-index-of-the-first-occurrence-in-a-string/)
- **核心定位**：原汁原味大海捞针。给定一极巨篇大长幅 $T$，给定一靶心签子 $P$。请还报出第一次重叠套牢时的开盘位点。
- **解题模板 (KMP 双轨并驱极速法)**：
  千万别用 `for-for` 这种在极限恶心 `AAAAAAAAAB` 长针上会被疯狂打崩。强迫手搓 KMP。
  **第一阶段造路引册（极其反人性但优雅的三行循环）**：
  建构神极字典一维表格 `pi（next）`：利用靶子**自己同自己比较去推演自己的宿命**！
  设置首位 `pi[0]=0`。游标从 1 开始跑向屁股尽头。同时拉扯一个由于失败向回跌落滚动的记录游标 `j`！如果此时前后追溯字符对眼了，`j++`！如果不对？那说明长条被斩断了！那极度冷酷无情的查去往事去跌落：一直向下面查本字典找 `j = pi[j-1]`！这正是 32.4 最极其核心的退避降阶求后路算法。
  **第二阶段长枪大破网**：
  把靶子带着建好的神引书拿去向巨大的文章海开始挑战穿梭。这和刚才那一步几乎代码极其复刻对偶对称一样！遇到挫折直接 `j = pi[j-1]` 一度向下退级回落保命求容身之阶。对上则极步向后跃升直到 $j == P.length()$ 功德圆满完成截获通杀返回端点。完全 $\Theta(M+N)$ 秒破。

## 2. 巧夺天工：字符串的内敛复制周期 (Easy)

`next / pi` 数组这个蕴藏前后一致的“神奇备忘录”能洞穿所有伪装。

### [LeetCode 459. 重复的子字符串 (Repeated Substring Pattern)](https://leetcode.cn/problems/repeated-substring-pattern/)
- **核心难局**：极其变态的一道极短极难破思路题。只给你一个长串比如 `abcabcabcabc`。你凭着火眼金睛能看出来特么的它是被 `abc` 这个子底柱积木通过复制连续四变串联大合成的吗？
- **KMP 解密流光法**：如果你拿着这个巨星字符串走一趟上面 KMP 的极其伟大的 $\pi$ 求真理数组法得到一个全长神记录数字字典！
  在记录集的最后一位上，那个写满了整套文本极境之最值的前后缀大数字 `pi[N-1]`，它代表了有**多么不可思议多大的一段极长前后印记是完全重叠影分身的**。
  如果我们把这整长身躯全剥离掉这部分长达几乎全身印记的重叠分身：**只剩下极其微小不可侵犯的那个孤寡小头块！** ($N - \pi[N-1]$ 这是极度奇迹所在！)。
  **神判法则**：如果这个巨无霸身躯的总长 $N$，竟然能够被这个剥离剩出来的微小原主初端骨架给整点完全没有余数的**无情除尽取模 (`N % (N - pi[N-1]) == 0`) 且最尾真皮子值居然不是 0 时**，就宣告了整个天地巨物必然是由这个极光微架构骨架大轮回应重覆制拼接生成的破灭真理！极致华丽极其优雅！这就是深挖 KMP 在周期重复探险的顶级绝杀应用。

 *(注：本题也有一种极其聪明的偷懒法，把字符串和自己原封不动重头缝起来组合成 `s + s`，如果你在这漫长两排海里强行去头掐烂尾还能在这海腰间再找到一次原汁极整的自己全须全尾原版，那它也是大叠怪！这是非底盘破圈的高阶巧劲)*。

## 3. 字符串与镜面世界极光碰撞：最强拼接 (Hard)

这是把求前后一致的神奇 `PI` 前缀函数推到终点的试金石魔题。

### [LeetCode 214. 最短回文串 (Shortest Palindrome)](https://leetcode.cn/problems/shortest-palindrome/)
- **题目神坑**：极其硬脾气变态！给你长长破字符串 `aacecaaa`。极其死板的苛求：**只允许你在最原始的前头疯狂塞字符！这要求你在塞足了料以后逼迫这个整个庞大杂合体首尾读念必须做到完全大镜面对称的回文图腾**。因为你加得越长越简单（比如把它翻边大翻转加塞），可是老子强要这是这个宇宙内里必须是改加了极其最短塞满的最短的图组合！你到底只能用倒过来的原字中的哪几位塞前脸！
- **真神归位 KMP**：这简直是最极境的“最长公共前后大冲波”。极其疯狂的奇思妙想构造：**我们原汁原味地把极其原本原装串 $s$，和在中间加上极其不沾边、阻断任何相接错串可能的一块大盾墙硬切神隔栏（比如特加个冷门字符 `#`），再去加上它全部原样大翻转后的极速镜面串 `reverse(s)` 进行拼接重置巨串：**
  $newString =$ `s` + `#` + `reverse(s)` ！
  拿着这个宏大无比极其奇特长造物，我们去极力无情调用我们在 28 题手搓的神兵宝具 KMP 缔造字典查 $\pi_{last}$ 那最后一个格的最高级记录数字！
  这在世界宣告什么！它所算出并且能完全跨过极度深渊长洋在逆天世界大彼岸竟然完全共振长相连的数字长！这说明在这个极其神秘数字长度下的前半最极点处的一段原身本子串是特么极其坚固的最长全包回文本色！
  那剩余的被遗弃在这一截完美回光镜像之外并且极其不规则的臭长后尾巴？那就是必须要反向抄袭并给大刺刺倒灌粘贴填在第一行最首位的债！这就是利用 KMP 把这种最丑恶回文拼接打服打进 $O(N)$ 直线时差线里！
