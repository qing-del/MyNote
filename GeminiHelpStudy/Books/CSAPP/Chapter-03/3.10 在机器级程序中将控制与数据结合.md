## 指针的力量深渊：越界穿透一切！

当你能以 `(%rax, %rbx, 4)` 之等随心调度的全盘无锁控制方法，去极具强大且丝毫不受规限管制地在浩渺的全部全能内存域（包含对指令控制中极其致命的调用返回流等进行极其重手大权限的复写覆改动作），这也诞生了系统领域万恶且最大的深渊源头破门攻击缝隙区！！
在 C 之中，指针根本不被约束也缺乏任何对它的校验，它对于跨距的极度任性随意游走使得原本是存储自身不连续大批量临时数值数组栈被轻松冲破边制横向跨流直打其致命死点所在（如前面 3.7 学过留存在栈中留用回退控制大旗用到的那个老 `call 返回起始地址 %rip 回溯点` 被更改向错误和不归路的黑洞之中去全开并移交整个大系统操作主控制全门！）。

## 缓冲区溢出与毁灭打击 (Buffer Overflow)

想象这样一个最为简单常用的读取外界并将其放入这极短极小的缓冲区组：
```c
void echo() {
    char buf[8];  // 仅仅八个孔位
    gets(buf);    // 非常出名极具全不加过滤控制只问结果不知长宽无节操要多少从网端取多少进这区间的获取器
    puts(buf);
}
```
**如果用户给你发了一长串极超过 8 个字长的代码？它会怎样？**
- 最简单的死相：越界跨出那个属于它安宁无用安全界内的组！去往下盖并直接擦干且替换成这个攻击者打在内线里面的特定恶意指令位置编码：去复写由于 `echo()` 这种需要由栈提供的向后进行函数脱离的返回起始点地标地址（老返回占位的指针向）！它可能被替换直接强制空降回到一段去调启恶意程序的黑指令集端从而被黑客劫走控制并完成木马后门等长植控制夺权。
- 也就是全美最大出名的 **"蠕虫（Worm）病毒黑风暴之源头"**。只因为一处缺乏检测与极其微小跨界长留缝隙将一门系统安全拉垮毁灭极境去。

## 对抗机制 - 第一道长城：地址空间布局随机化 (ASLR)

过去的时代！一旦发现这个系统的版本可以越位拿进这种攻击控制地址，只要攻击者拿着同样的这个地址全网投毒所有的带出机器即可瞬间让黑客如过无人境。为什么？因为之前大家都按统一且静态长存一致的位配分布全同布排开（栈就是放那个地方）。
**地址空间布局随机化（ASLR, Address-Space Layout Randomization）** 阻绝！
- 现在你即使写死一个要指向黑进去破坏极长执行地的地址。但每进行新跑拉起，那一大批所有重要命脉的极其极段位置及函数所分部位置全都经过了超级随心打乱，进行大极值偏离！要让这套系统攻击的地址成功概率被严重地压缩至完全没有盲抓和猜点极高准确可行性的微小不可查成功几率里！

## 对抗机制 - 第二道屏障：栈破坏检测 (金丝雀机制 Canary)

这门技术基于及其聪明简要且极其易用的设计被大量且直接集成到所有当前主流最高级 C / GCC 中提供。
> 类比极妙故事：在矿工为了能活下去在充满极高度毒瓦斯煤矿干活中会在身下养极其极为娇脆对于极其轻微微毒全倒下的那一只 “金丝雀”。如果没看到或发现倒地不起立马所有极速拉起停掉全关。这就叫做全天候高极高灵敏感警装置！

在计算机里：编译器在为这段极高风险函数构建生成进入最初栈模型中（分配在局部安全极危变量与那个神圣需护的要退出的返回长指针地扯夹着的那一层）：它放且安防进一道非常严密不外外露不能改置的一个随机产生乱极序列的极其难预测密码词——**也就是金丝雀 Canary**！
- 如果溢出者从上面的弱制向这边横推并漫步下落淹且覆过来以试图掩藏直打最后老基底的那一片地方。他必定必然绝不可能不覆掉打断擦黑这放在此间的这一极其极密的不可查代码并改变产生碎杂碎破局。
- 退出执行脱离返回全流出最底退关大底动作之时它要过一道检验局（把其最初那个存此之印再次极其重核对比此门存单）。如果是改毁或换断：全跑掉立刻截门报极死罪断点截断中止：也就是出名的 "**Stack Smashing Detected**" ！强制保护防止了最坏执行转手失陷交替可能局步段！。

## 对抗机制 - 终极堡垒：限制极度危险的可执行代码区域

以前的旧黑客极爱于把你申请下放入去接数据的空栈以及从外取进大字符串数组数据填在那个段区域！并且通过直接回写引导指令冲指到那个你取了这帮脏大长数字数据的极底处执行（因为它把那里写满被带进了全是其用恶长机编码去接去改的大杀控制黑件去直调拉起底层核心操作级指令）。
如今系统（配合极其极速强大之全新硬件底层位保护加持 `NX/XD-bit` 强截拦）：
- 直接拉一根绝线：这栈和这些申请的只能供你用于数据使用放拿（`Read+Write`），但不容许这从外界大取这区域具备带向能够跑拉 `执行(Execute)` 全起能力！！把越界的死控制执行企图这火硬扑绝在全限控制外层！使得把外部乱接码拉下到去让这段产生长管流大变幻完全杜禁被限。
