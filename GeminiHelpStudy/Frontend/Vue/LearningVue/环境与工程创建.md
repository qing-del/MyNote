### 🛠️ Step : 环境准备 (Node.js)
要开发 Vue 3，我们需要前端的运行环境 Node.js。
请打开命令行（CMD 或 PowerShell），输入：

```Bash
node -v
npm -v
```

> [!info] npm的全称为`Node Package Manager` -- `NodeJS`的软件包管理器
- **如果显示版本号**（例如 `v18.x.x` 或更高）：说明 **"Node Ready"**。
- **如果报错**：请去 [Node.js 官网](https://nodejs.org/) 下载 **LTS 版本**（长期支持版）并安装。

### 💻 基本指令

#### 引入新的工具包
- 若是后续需要加入`Xxxx`工具包，可以利用以下指令
- 执行完毕之后就会安装到对应项目的`node_modules`文件夹中
- 
```Bash
# 安装 Xxxx 软件包
npm install Xxxx
```
> [!info] 
> 在`Xxxx`后面加入`@x.x`说明下载 x.x 版本的 Xxxx 软件包
> 在后面加入 `--save` 会保存到 `package.json` 当中
> 例如：`npm install Xxxx@x.x --save`

#### 配置全局安装路径
- 以下是示例，路径记得要换成`NodeJS`的安装目录
```Bash
npm config set prefix "D:\develop\NodeJS"
```
> [!Error] 注意：示例中的`D:\develop\NodeJS`是`NodeJS`的安装目录

#### 仓库切换
- 如果平时下载工具速度较慢，可以切换为淘宝镜像
- ```Bash
  npm config set registry http://registry.npmmirror.com
  ```

### 🛠️ Step 1: 创建前端项目 (脚手架)
不再使用老的 `vue-cli`，而是使用更快的 **Vite**。
请在你存放 `task-system` 后端项目的**同一级目录**（不要在后端文件夹里面），打开终端，执行以下命令：
```Bash
npm create vue@3.3.4
```

**执行过程中会问你几个问题，请按以下选择（用键盘上下键移动，回车确认）：**
![[vue项目构建选项.png]]

#### `vue ui`面板
- 在cmd命令行中输入此命令回车，会弹出一个连接URL
- 点击之后会进入到`vue ui`的面板，可以使用面板来管理和创建脚手架
```bash
vue ui
```

### 📦 Step 2: 安装“三件套” (依赖)
项目创建好后，我们需要安装三个关键库：
1. **Axios**: 前端的 `HttpClient`，用来给你的 Spring Boot 发请求。
2. **Element Plus**: 类似 Java Swing/JavaFX 的组件库，有一堆现成的漂亮按钮、输入框。
3. **Vue Router**: 负责页面跳转（登录页 -> 游戏主页）。

请依次执行以下命令：
```Bash
# 1. 进入项目文件夹
cd lifegame-ui

# 2. 安装基础依赖 (相当于 maven reload)
npm install

# 3. 安装我们的武器库
npm install axios element-plus vue-router
```
- 第二条命令之后，项目中会多出`node_moudules`文件夹 -- 存放依赖

#### 基本项目结构
- **Vue项目**
	- `node_modules`文件夹 -- 下载的第三方包
	- `src` -- 源代码
		- `assets` -- 静态资源目录：存放图片、字体
		- `components` -- 组件目录，存放通用组件
		- `App.vue` -- 根组件
		- `main.js` -- 入口文件
	- `package` -- 项目配置文件，包括项目名、版本号、依赖包、版本等
	- `vite.config.js` -- Vue项目的配置信息，如`端口号`等

### 🧹 Step 3: 清理门户 (删除样板代码)
Vite 默认会生成一个“Hello World”页面，我们需要把它删干净，只留一张白纸。
请用你的 IDE (IntelliJ IDEA 或 VS Code) 打开 `lifegame-ui` 文件夹，做以下删减：
1. **删除** `src/components/HelloWorld.vue` (直接删文件)。
2. **删除** `src/style.css` 里的所有内容（清空文件，但保留文件）。
3. **修改** `src/App.vue`，把里面的内容**全部删掉**，替换为以下最简代码：


```HTML
<script setup>
  // 这里写 JS 逻辑
</script>

<template>
  <div id="app">
    <h1>🚀 LifeGame 前端已启动</h1>
  </div>
</template>

<style>
  /* 全局样式重置 */
  body { margin: 0; padding: 0; background-color: #f0f2f5; font-family: sans-serif; }
</style>
```

### 🚀 Step 4: 启动！

在终端输入：

```Bash
npm run dev
```

你会看到一个类似 http://localhost:5173 的地址。

按住 Ctrl 点击这个链接，如果浏览器里出现了一行大字 “🚀 LifeGame 前端已启动”，那就说明环境搭建完美成功！
